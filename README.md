# PHP MongoDB Authentication System

##  Project Description
This project is a simple authentication system built using PHP and MongoDB.
It includes user signup, login, dashboard, and logout functionality.

---

## Features
- User Registration (Signup)
- User Login
- Dashboard Page
- Logout System
- MongoDB Database Connection

---

## Technologies Used
- PHP
- MongoDB
- Composer
- HTML
- XAMPP

---

## Setup Instructions

1. Install XAMPP and start Apache.
2. Install MongoDB and ensure it is running.
3. Clone this repository:

   git clone https://github.com/your-username/I_MONGODB.git

4. Navigate to project folder:

   cd I_MONGODB

5. Install dependencies:

   composer install

6. Configure .env file with your MongoDB connection string.
7. Place the project inside htdocs folder (if using XAMPP).
8. Open in browser:

   http://localhost/I_MONGODB/
